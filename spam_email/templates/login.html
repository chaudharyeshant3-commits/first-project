⚙️ One-Time Google Cloud Setup

  1. Go to console.cloud.google.com
  2. Create a new project (e.g. MailGuard)
  3. Enable Gmail API from API Library
  4. Go to APIs & Services → Credentials
  5. Create OAuth 2.0 Client ID → Web Application
  6. Add redirect URI: http://localhost:5000/oauth2callback
  7. Download JSON → rename to credentials.json → place in project folder
  8. Restart the server and click Connect with Gmail!
